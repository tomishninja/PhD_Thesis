The placement accuracy of the virtual icosahedron object was also analyzed with respect to the user's viewpoint at the time of placement. To compute this, the actual and correct placement positions were transformed using the headset's position and orientation matrix, then we calculated the distance between both positions.
For the direct viewpoint accuracy, the model showed a significant fixed effect of the x-ray visualization effects ($\chi2$(4, N=20) = 50.314, p  $<$ 0.0001)
and the presence of reference objects ($\chi$2(1, N=20) = 47.777, p $<$ 0.0001) and a significant interaction effect ($\chi$2(4, N=20) = 18.428, p = 0.001).

Post-hoc pairwise comparisons on the presence of reference objects showed a significant improvement in accuracy when reference objects were present (p$<$0.0001, df = 3018, t = -6.662). Comparisons between the different visualizations showed a significant improvement in accuracy between Edge and None (p = 0.0003, df = 989, t = -4.145), Random Dot (p $<$ 0.0001, df = 989, t = -6.883), Saliency (p $<$ 0.0001, df = 989, t = -5.021) and Tessellation (p = 0.0006, df = 989, t = -2.669). Variation was observed, but no significant difference found was found between Random Dot and None (p = 0.0589, df = 989, t = -2.662), and Random Dot and Tessellation (p = 0.0507, df = 989, t = -2.725).
Significant pairwise comparisons between visualization and the presence of the reference objects showed various significant differences outlined in \autoref{tab:view-comparisons}.

\paragraph{Placement Accuracy by Axis:} \label{sec:X-ray Quantitative Results Placement Accuracy}

    The distance away from the goal of the objects was based on their position relative to the box.

\end{figure*}

% X axis
The placement accuracy of the virtual icosahedron object was also analyzed on each axis to identify any effects of the physical box orientation. On the x-axis of the box, the model showed a significant fixed effect of the presence of reference objects  ($\chi$2(1, N=20) = 16.221, p $<$ 0.0001). No significant fixed effect of the X-ray visualization effects, 
or interaction effect was found.
The post-hoc pairwise comparisons showed the presence of the reference objects produced a significantly lower accuracy on the x-axis of the box (p $<$ 0.0001, df = 989, t = -4.723). 

% Y axis
On the y-axis of the box, the model showed a significant fixed effect of x-ray visualization effects ($\chi$2(4, N= 20) = 11.828, p = 0.0187) on accuracy. The fixed effect of the presence of the reference objects showed some variation but was not significant ($\chi$2(1, N= 20) = 3.695, p = 0.0546), and no significant interaction effect was found. 
Post-hoc pairwise comparison of y-axis accuracy between x-ray visualizations showed no significant effects. 

% Z axis
On the z-axis of the box, the model showed a significant fixed effect regarding x-ray visualization effects $\chi$2(4, N= 20) = 10.1573, p = 0.0379, and the presence of the reference objects $\chi$2(1, N= 20) = 5, p = 0.0254. No significant interaction effect was found.
Post-hoc pairwise comparisons on the presence of reference objects showed significantly improved accuracy on the z-axis (p = 0.0263, df = 989, t = -2.226) when reference objects were present. Comparisons between X-ray visualizations showed no significant differences.
